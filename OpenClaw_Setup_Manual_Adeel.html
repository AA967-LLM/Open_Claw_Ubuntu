02. Ubuntu OS Hardening & Prep

OpenClaw requires specific C++ and Python headers to compile its native modules (like node-gyp for SQLite/WhatsApp).

  • 1 Update & Essential Build Tools: 
    sudo apt update && sudo apt upgrade -y
sudo apt install -y build-essential python3-dev libpython3-dev \
pkg-config libcairo2-dev libpango1.0-dev libjpeg-dev libgif-dev \
librsvg2-dev ffmpeg git curl
  • 2 Node.js Installation (LTS): Always use nvm or fnm to manage Node versions. Avoid sudo apt install nodejs. 
    curl -fsSL https://fnm.vercel.app/install | bash
fnm use --install-if-missing 22

03. Core OpenClaw Installation

  • 1 CLI Installation: 
    npm install -g openclaw
  • 2 Gateway Auto-Start (systemd): Create a service to ensure OpenClaw wakes up when the machine restarts. 
    # Create file at /etc/systemd/system/openclaw.service
[Unit]
Description=OpenClaw Gateway Service
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/root/.openclaw/workspace
ExecStart=/usr/local/bin/openclaw gateway
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
    Enable with: sudo systemctl enable --now openclaw
Performance Tuning

04. Efficient "Brain" Routing

To keep costs zero and performance high, we use **ClawRouter** logic to prioritize local models over paid APIs.

Context Window Rule: Always ensure your primary conversation model has at least 16,000 tokens. Local models like Step3-VL-10B (4k window) should only be used as Sub-Agents for vision/reasoning.

Preferred Routing Matrix:

  • Daily Talk: ollama/deepseek-v3.1:671b-cloud (Infinite window/Free)
  • Vision/Images: step3-local/Step3-VL-10B (Running on your RTX 4080)
  • Complex Code: anthropic/claude-3.5-sonnet (Fallback only)

05. Mastering the D: Drive Mount

For WSL2 setups, your Windows LLM inventory is accessible via /mnt/d/. The model loader must be configured to use these absolute paths.

# In your openclaw.json:
"models": {
  "step3-local/Step3-VL-10B": {
    "localPath": "/mnt/d/Google antigravity/ai-stack-2026/Step3-VL-10B-AWQ",
    "device": "cuda"
  }
}

06. Persistent Memory (memU)

To ensure Open Claw "remembers" your Win11 environment across restarts:

  • Maintain SOUL.md for behavioral rules.
  • Use USER.md for personal context (Adeel, RTX 4080, D: drive structure).
  • Periodically run openclaw memory reindex to keep the vector store fresh.
